# 墨羽-自适应记忆工具包

> 给你的 AI Agent 装上 16 大记忆能力，跨会话真正记得你是谁。零平台绑定，开箱即用。

GitHub 仓库：https://github.com/awchzh/moyu-memory

## 16 大能力 — 按领域分组

### 📦 记忆层 — 存储与检索

| # | 能力 | 说明 |
|---|------|------|
| 1️⃣ | **TEMPR 多策略检索** | 语义 + BM25 关键词 + 时间加权 + RRF 融合排序 |
| 2️⃣ | **工作记忆** | 独立文件存储，上下文压缩也不丢；内建工作流知识库检索（`moyu kb search`） |
| 3️⃣ | **轻量知识图谱** | 自动提取实体关系。基础模式（正则，无需 API），配置 key 后语义深度提取 |
| 4️⃣ | **用户画像** | 自动从对话中提取偏好和事实。基础模式（正则，无需 API），配置 key 后语义深度提取 |
| 5️⃣ | **防重复** | SHA256 哈希去重，不会重复存储 |

### 🧠 学习层 — 从互动中进化

| # | 能力 | 说明 |
|---|------|------|
| 6️⃣ | **从纠正中学习** | 识别"不对/不要/记住"信号，3次晋升为永久规则，同类错误不再犯 |
| 7️⃣ | **自我反思** | 醒来时自动分析旧记忆，发现关联与矛盾 |

### 🛡️ 防御层 — 保护与验证

| # | 能力 | 说明 |
|---|------|------|
| 8️⃣ | **完整性校验** | 启动时自动校验 + 手动 `moyu check`，SHA256 比对静态文件 |
| 9️⃣ | **自动备份与恢复** | 每天一次全量备份，保留最近 3 天。备份与校验独立运行，互不阻塞 |
| 🔟 | **法医分析** | 检测篡改文件中是否包含指令覆盖/角色改写/规则注入等攻击关键词及 JSON 结构异常 |
| 1️⃣1️⃣ | **记忆自我保护** | 密码验证。危险操作需密码确认，3 次失败自动锁定 30 分钟 |

### 🌿 生命周期层 — 让记忆会呼吸

| # | 能力 | 说明 |
|---|------|------|
| 1️⃣2️⃣ | **上下文自适应压缩（两级渐进）** | 占用率超过 70% 温和截断长记忆，超过 85% 激进降级非关键项。压缩可回溯——原文保存到 `refs/`，`moyu ref <名称>` 随时查看完整原文 |
| 1️⃣3️⃣ | **遗忘曲线（三阶段闸门 + 场景保护）** | 14天安全窗口 + 访问密度趋势分析 + 场景关联保护。热点场景下全部相关记忆不解降级。支持自定义参数：`moyu forget set` |
| 1️⃣4️⃣ | **场景标签** | 三优先级：墨白手动标注 > 用户自定义（`config.yaml` 配置） > 关键词自动提取。永不覆盖手动标注 |
| 1️⃣5️⃣ | **记忆合并** | 检测关键词重叠的相关记忆，自动合并为一条综合摘要。原始细节存入可展开字段 |
| 1️⃣6️⃣ | **自动更新** | 自动检测 GitHub 新版本，一键升级——保留 memory_data 和用户配置 |

## 快速开始

```bash
pip install numpy requests pyyaml
```

把 `moyu_toolkit/` 文件夹复制到你的项目中：

```bash
cd moyu_toolkit
python3 agent_memory.py search "我们聊过什么"
```

**一条命令统领全局：**

```bash
cd moyu_toolkit && python3 moyu.py help
```

## 命令参考

```bash
cd moyu_toolkit && python3 moyu.py <命令> [参数]
```

### 🛡️ 安全

| 命令 | 说明 |
|------|------|
| `moyu setup` | 设置安全密码 |
| `moyu verify <类型> [描述]` | 验证危险操作 |
| `moyu unlock` | 解锁安全系统 |
| `moyu check` | 完整性校验（SHA256） |
| `moyu audit` | 三层防御链汇总 |
| `moyu init` | 初始化校验清单 |

### 🧠 记忆与检索

| 命令 | 说明 |
|------|------|
| `moyu search <关键词>` | TEMPR 多策略检索 |
| `moyu stats` | 全统计 |
| `moyu status` | 系统状态 + 防御链可视化 |
| `moyu inject` | 获注入行为规则 |
| `moyu signals` | 查看学习触发词 |

### ⏳ 生命周期与压缩

| 命令 | 说明 |
|------|------|
| `moyu forget` | 遗忘曲线状态（三阶段闸门） |
| `moyu forget config` | 查看遗忘曲线参数 |
| `moyu forget set <键> <值>` | 设置参数：`demote_days`, `archive_days`, `min_keyword_length` 等 |
| `moyu forget scene labels` | 在 config.yaml 设自定义场景标签 |
| `moyu compress` | 查看压缩状态 |
| `moyu compress --now` | 手动压缩 |
| `moyu compress config` | 查看压缩参数 |
| `moyu compress set <键> <值>` | 设置参数：`mild_threshold`, `auto_threshold` 等 |
| `moyu context` | 上下文占用率概览 |
| `moyu ref <名称>` | 读取压缩前原文（可溯源） |
| `moyu ref list` | 列出所有 refs |

## 版本 v2.1.2 变更

- **场景保护（新功能）** — 遗忘曲线第三阶段：热点场景下全部相关记忆不解降级。场景标签三优先级（手动标注 > 用户自定义 > 关键词自动提取）
- **两级渐进压缩** — 70% 温和截断 / 85% 激进降级，参数可调
- **可溯源压缩** — 压缩前原文保存到 `refs/`，`moyu ref <名称>` 随时查看完整原文
- **RRF 融合检索** — Reciprocal Rank Fusion 替代加权平均，搜索结果更稳定
- **备份修复** — 每日备份不再被完整性校验阻塞，独立运行
- **完整命令参考** — 生命周期与压缩自定义参数、可溯源命令、场景标签说明等
